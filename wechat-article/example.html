微信公众号自动化发布完全指南

你是否还在为公众号发文而烦恼？手动排版、逐张上传图片、反复粘贴到后台……这些重复劳动完全可以自动化。

本文介绍 wechat-toolkit — 一个专为微信公众号设计的自动化发布工具包，让你用 Markdown 写作，一键发布到公众号。

为什么需要自动化？

手动发布公众号文章的痛点相信你深有体会：

  • 排版耗时：在编辑器里反复调整样式
  • 图片麻烦：一张张上传，还要处理尺寸
  • 流程繁琐：复制粘贴、预览、修改、再预览
  • 格式丢失：从 Markdown 粘贴后样式全乱

自动化不是偷懒，而是把时间花在真正重要的事情上——内容创作。

核心功能一览

wechat-toolkit 提供三个核心能力：

快速上手

第一步：配置凭证

在开始使用前，需要设置微信公众号的 API 凭证：

bash
export WECHAT_APP_ID=your_app_id
export WECHAT_APP_SECRET=your_app_secret

获取方式：登录微信公众平台（mp.weixin.qq.com）→ 开发 → 基本配置。

第二步：编写文章

使用熟悉的 Markdown 语法撰写文章。支持标准的标题、段落、列表、代码块、表格等语法。

第三步：AI 智能排版

将 Markdown 文件交给 AI，自动转换为适配微信公众号的 HTML：

  • 所有样式内联（inline style）
  • 手机端优先的阅读体验
  • 代码块语法高亮
  • 自适应图片宽度

第四步：一键发布

bash
# 发布到草稿箱
node ./scripts/publisher/publish.js article.html

脚本会自动完成：上传内联图片 → 生成封面 → 提取摘要 → 发布到草稿箱。

技术细节

图片处理机制

发布脚本会自动扫描正文中的本地图片路径，上传到微信永久素材库，并将 URL 回填替换。整个过程无需手动干预。

支持的处理流程如下：

  1. 1解析 HTML 中所有 <img> 标签
  2. 2判断 src 是否为本地文件路径
  3. 3调用微信 API 上传图片
  4. 4用返回的 URL 替换原始路径
  5. 5输出最终可发布的 HTML

封面图策略

封面图通过 frontmatter 的 cover 字段指定：

yaml
---
cover: /absolute/path/to/cover.jpg
---

如果未指定封面图，脚本会自动生成一张 900x500 的蓝色默认封面，确保不会因为缺失封面而发布失败。

常见问题

Q：IP 不在白名单怎么办？

运行以下命令查看当前公网 IP：

bash
curl ifconfig.me

然后将输出的 IP 地址添加到微信公众号后台的白名单中。

Q：支持哪些图片格式？

目前支持 jpg、jpeg、png、gif、bmp、webp 六种格式，单张图片大小不超过 10MB。

Q：永久素材库有上限吗？

有，微信永久素材库的图片上限为 5000 张。建议定期清理不再使用的图片资源。
