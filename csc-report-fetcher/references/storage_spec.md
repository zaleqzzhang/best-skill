### 标题简称规则

- 去掉引号、冒号等特殊字符
- 长度控制在20个汉字以内
- 保留核心关键词
- 用下划线替代空格

### 元信息字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 来源 | 是 | 固定为"中信建投证券研究" |
| 转载站 | 是 | 实际抓取的网站名 |
| 原始链接 | 是 | 转载页URL |
| 分析师 | 否 | 如文中提及则记录 |
| 发布日期 | 是 | YYYY-MM-DD |
| 行业分类 | 是 | 见上方标签表 |
| 报告类型 | 是 | 行业深度/行业跟踪/宏观点评/策略研究/政策研究 |
| 正文完整度 | 是 | 完整正文 / 摘要版 |
| 抓取时间 | 是 | YYYY-MM-DD HH:MM |
