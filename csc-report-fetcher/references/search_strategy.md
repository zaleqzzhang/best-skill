# 中信建投研报搜索策略详解

## 核心发现

中信建投证券研究公众号每天早上发布一篇"今日投资精华"汇总贴，将当天推荐的3-5篇研报核心观点浓缩在一起。这篇汇总贴会被腾讯新闻、同花顺等多家财经媒体转载，且转载版是完整正文。

## 搜索关键词模板

### 必搜（每日汇总）

```
"中信建投" "今日投资精华" {日期}
```

这是最有价值的单次搜索，能拿到当天所有研报的核心观点汇总。

### 补充搜索（单篇研报）

```
"中信建投" 研报 {YYYY-MM-DD}
"中信建投" 研报 {YYYY年M月D日}
site:news.qq.com "中信建投" {YYYY-MM-DD}
```

### 特定行业搜索（按需）

```
"中信建投" 光纤 {日期}
"中信建投" AI算力 {日期}
"中信建投" 医药 {日期}
```

## 日期格式处理

搜索引擎对日期的识别不统一，建议同时使用多种格式：

- `2026-04-16` — 标准ISO格式，最常用
- `2026年4月16日` — 中文全格式，搜索引擎友好
- `4月16日` — 短格式，匹配更多结果但可能有历史干扰
- `20260416` — 紧凑格式，部分URL包含

## 常见文章类型

中信建投每天通常发布以下类型的文章：

1. **今日投资精华**（每日固定）— 3-5篇研报的核心观点汇总
2. **行业深度**（不定期）— 某个行业的深度分析报告
3. **宏观点评**（高频）— 经济数据发布后的点评
4. **策略周报/月报**（定期）— 市场策略与配置建议
5. **政策点评**（事件驱动）— 重大政策出台后的分析
6. **新股策略**（不定期）— IPO/再融资相关分析

## 故障排除

### 搜索结果为空
- 尝试调整日期格式
- 移除 site: 限定，搜全网
- 可能是非工作日，中信建投周末/节假日不发报告

### web_fetch 返回空内容
- 部分网站有反爬，换一个转载站
- 腾讯新闻最稳定，优先用

### 文章内容不完整
- 检查是否是摘要版转载
- 在元信息中标注 `正文完整度: 摘要版`
- 尝试从其他转载站获取同一篇文章的完整版
