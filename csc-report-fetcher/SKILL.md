# 中信建投研报自动拉取 Skill

> **一句话概述**：自动搜索并抓取中信建投证券研究每日发布的所有行业研报，存储为结构化 Markdown 文件。

## 背景

中信建投证券研究官网研报系统完全封闭（需登录/付费），但其公众号文章会被多家财经网站转载，且转载版可获取完整正文。本 Skill 利用这一特点，通过搜索转载站来获取研报内容。

## 核心流程

```
Step 1: 搜索当天文章
    ├── web_search: "中信建投" "今日投资精华" + 当天日期
    ├── web_search: "中信建投" 研报 + 当天日期（YYYY-MM-DD / YYYY年M月D日）
    └── web_search: site:news.qq.com "中信建投" + 当天日期
    
Step 2: 抓取完整正文
    ├── 优先: news.qq.com（腾讯新闻）→ 完整正文概率最高
    ├── 备选: 10jqka.com.cn（同花顺）→ 科技/行业类完整度高
    ├── 备选: k.sina.com.cn（新浪财经）→ 深度研报全文好
    └── 兜底: nbd.com.cn / 21jingji.com 等 → 通常只有摘要

Step 3: 内容筛选（判断是否为行业分析研报）
    ├── ✅ 保留: 有明确行业/板块分析、数据支撑、投资逻辑的深度报告
    ├── ✅ 保留: 宏观经济分析、政策研究、策略研报
    ├── ✅ 保留: 新股/IPO策略分析
    ├── ❌ 排除: 纯广告、基金公告、个股快讯
    └── ❌ 排除: 重复转载版本（同一研报只保留一份）

Step 4: 存储到目标路径
    └── {inbox_path}/{YYYY-MM-DD}/{序号}_{行业}_{标题}.md
```

## 使用方式

### 基本用法

当用户说以下类似的话时，触发此 Skill：

- "拉一下今天中信建投的研报"
- "把中信建投今天的文章抓下来"
- "帮我拉取今天的券商研报"
- "中信建投研报入库"

### 执行步骤

#### Step 1: 确认日期和存储路径

```
默认日期: 当天
默认路径: /Users/yeshidu/Desktop/code/shidu_code/xhs-manager/backend/data/reports/inbox/
```

如果用户指定了不同的日期或路径，以用户为准。

#### Step 2: 搜索文章

使用 `web_search` 工具，按以下策略搜索：

```
搜索1: "中信建投" "今日投资精华" {日期}
  → 拿到每日汇总贴（含3-5篇研报核心观点，最有价值）

搜索2: "中信建投" 研报 {YYYY-MM-DD}
  → 拿到当天所有单篇研报

搜索3: site:news.qq.com "中信建投" {日期}
  → 补充腾讯新闻转载版
```

**重要**: 不要限定行业关键词！搜全行业，中信建投每天覆盖多个行业。

#### Step 3: 抓取正文

对搜索到的每篇文章，使用 `web_fetch` 工具读取完整正文：

```
优先级:
1. news.qq.com/rain/ → 腾讯新闻转载，完整正文概率 >90%
2. news.10jqka.com.cn → 同花顺转载，科技类文章完整度高
3. k.sina.com.cn → 新浪财经，深度研报全文
4. finance.eastmoney.com → 东方财富
5. 其他站点 → 通常只有标题+摘要
```

#### Step 4: 内容判断

逐篇阅读内容，判断是否为行业分析研报：

| 类型 | 判断 | 说明 |
|------|------|------|
| 行业深度报告 | ✅ 保留 | 有完整数据分析、行业逻辑 |
| 宏观经济分析 | ✅ 保留 | GDP/CPI/出口等宏观数据分析 |
| 策略研究 | ✅ 保留 | 市场策略、板块配置建议 |
| 政策点评 | ✅ 保留 | 政策影响分析，有数据支撑 |
| 新股/IPO策略 | ✅ 保留 | 上市标准分析、打新策略 |
| 每日汇总精华 | ✅ 保留 | "今日投资精华"属于高价值汇总 |
| 纯广告推广 | ❌ 排除 | 基金销售、产品推荐 |
| 基金净值公告 | ❌ 排除 | 无分析价值 |
| 重复转载 | ❌ 排除 | 同一研报只保留最完整版本 |

#### Step 5: 写入文件

每篇研报保存为一个 Markdown 文件，格式如下：

**文件名规范**: `{序号}_{行业}_{标题简称}.md`

```
示例:
01_宏观_中国出口能否顶住高油价压力.md
02_新股策略_深化创业板改革支持未盈利IPO.md
03_非银金融_2025年证券行业年报综述.md
04_科技AI_Mythos与谷歌TPU产业链.md
```

**文件内容模板**:

```markdown
---
来源: 中信建投证券研究
转载站: {来源网站名}
原始链接: {URL}
分析师: {分析师姓名，如能获取}
发布日期: {YYYY-MM-DD}
行业分类: {宏观/策略/科技/农业/医药/非银/汽车/...}
报告类型: {行业深度/行业跟踪/宏观点评/策略研究/政策研究}
正文完整度: {完整正文/摘要版}
---

# {报告标题}

{正文内容}
```

#### Step 6: 创建目录并确认

```bash
mkdir -p {inbox_path}/{YYYY-MM-DD}/
```

写入完成后，列出文件树确认。

|------|------|

## 扩展能力

### 东方财富研报 API（辅助）

如需补充更多券商的研报，可调用东方财富 API：

```
URL: https://reportapi.eastmoney.com/report/list
参数:
  - industryCode: 行业代码（448=通信设备, 1036=半导体, *=全部）
  - qType: 1=行业研报
  - pageSize: 最大100
  - beginTime/endTime: YYYY-MM-DD
```

**注意**: 东方财富不覆盖中信建投等头部券商，仅有中小券商研报。

### westock-data 脱水研报（辅助）

通过 westock-data skill 的 `dehydrated` 功能可获取编辑精选研报的完整 HTML 正文，质量最高但不保证每天有特定行业。

## 注意事项

1. **搜索不限行业** — 中信建投每天发多个行业的文章，不要只搜某一个行业
2. **去重** — 同一篇研报可能在多个网站转载，只保留最完整的一版
3. **标注正文完整度** — 有些网站只转载摘要，要在元信息中标注
4. **腾讯新闻最优** — news.qq.com/rain/ 路径的转载完整度最高，优先使用
5. **日期格式多样** — 搜索时用多种日期格式：YYYY-MM-DD、YYYY年M月D日、M月D日
