# 分场剧本格式规范

## 场景题头格式

格式：`场号 场景名 [内/外] [日/夜]`

- 场号：x-x（第几集-第几场）
- 场景题头需要**加粗**

示例：**1-1 废弃仓库 [内] [夜]**

## 出场人物

每一集最前面列出当集出场人物的名称，**加粗**。

示例：**出场人物：林晓，妈妈**

## 画面/动作 (Action)

- 所有环境描写、人物动作、镜头画面，必须以符号"△"开头
- **严禁出现心理描写**（如"他心里想..."），必须转化为可视化的动作或表情

示例：
```
△ 陈墨推开门，掸了掸身上的雨水。
△ 林晓换鞋进门，将背包随手扔在沙发上，揉了揉眉心。
```

## 对白 (Dialogue)

采用"紧凑格式"，人名后加冒号，人名要加粗。

格式：`**角色名**：(情绪/动作提示) 台词内容`

示例：
```
**陈墨**：(冷笑) 你以为你能跑得掉？
**林晓**：(喝了口水，声音沙哑) 不了妈，没胃口。
**妈妈**：(轻声) 今天又加班到这么晚？饭热过了，要不要吃点？
```

## 特殊音效与视觉

- 重要的声音/音效：`【音效】远处传来警笛声。`
- 屏幕上的文字/字幕：`【字幕】三年前`

## 专业术语

| 术语 | 标注方式 | 说明 | 示例 |
|-----|---------|------|------|
| 画外音 | (O.S.) | 人在现场但镜头未拍到嘴动 | **陈墨**(O.S.)：别过来！ |
| 内心独白 | (V.O.) | 心理声音 | **陈墨**(V.O.)：哪怕只有百分之一的机会，我也要试一试。 |
| 闪回 | 【闪回】...【闪回结束】 | 回忆片段 | - |

## 格式注意事项

1. 不要把"【画面/动作 (Action)】、【场景题头】、【对白 (Dialogue)】"等分类字眼放到剧本里，只需有其内容
2. 每一句话之间都要换行
3. 遵守剧情路线

## 台词处理原则

大段的对白或独白，需要加工处理：
- 拆分：转换成角色间你一句我一句的对白
- 转化：转化为可见的戏剧动作，通过角色表演来传递信息

## 写作要求

1. **写作细节**：戏要一场一场写，时空转换需控制。每场戏的台词、动作、表情、内心状态、情绪气氛、场景道具等细节都要考虑
2. **规范格式**：简洁准确，用剧本语言写出画面感
3. **戏剧化冲突**：删减、合并、提纯、夸张矛盾冲突，转化为可见的外部动作冲突
4. **情绪钩子**：悬念、反常识、倒计时、身份反转、打脸、系统、危机、痛点

## 完整示例

```
第一集
**出场人物：林晓，妈妈**

**1-1 林晓家客厅 [内] [日]**
△ 林晓换鞋进门，将背包随手扔在沙发上，揉了揉眉心，径直走向饮水机接水。

△ 妈妈端着一盘水果从厨房走出。

**妈妈**：(轻声) 今天又加班到这么晚？饭热过了，要不要吃点？

**林晓**：(喝了口水，声音沙哑) 不了妈，没胃口。

△ 林晓坐到沙发上，拿起手机解锁。

**林晓**(V.O.)：要是当初没选这个行业，现在会不会轻松点？

△ 她闭眼深呼吸，眉头拧得更紧。

**画外音**(V.O.)：三年前的那个下午，正是这个决定，改变了她的生活轨迹。

【闪回】

**1-2 大学会议室 [内] [日]**
△ 林晓站在答辩台中央，手里攥着演讲稿。

**林晓**：我认为新媒体行业的发展潜力巨大，我希望能深耕这个领域。

**导师**：(点头) 有想法，但这个行业竞争激烈，你要做好吃苦的准备。

△ 林晓用力点头，眼神坚定。

【闪回结束】

**1-3 林晓家客厅 [内] [日]**
△ 林晓睁开眼，手机从手中滑落，发出轻微声响。

△ 妈妈走过来捡起手机，轻轻放在茶几上。

**妈妈**：(拍了拍她的肩膀) 累了就歇歇，妈永远支持你。

△ 林晓转头看向妈妈，眼眶一热，嘴角勉强扯出一丝笑。
```
