# 提示词模板与词库速查

## 一、人物三视图提示词模板

### 框架
```
【画风与风格定位】+【性别、年龄、职业/身份】+【身材与整体比例（适当夸张）】+【脸型与五官特征（突出记忆点）】+【发型与发色（标志性设计）】+【服装与配饰（强调标志性元素）】+【表情与气质（体现角色性格）】+【姿态（标志性动作）】+【画面规格与技术约束】
```

### 画面规格
- 生成全身三视图以及一张面部特写
- 最左边占满1/3的位置是超大的面部特写，右边2/3放正视图、侧视图、后视图
- 纯白背景，21:9横向宽幅构图
- 高精细线稿与上色，无道具，无其他人物，无文字

### 完整示例
```
生成全身三视图以及一张面部特写。(最左边占满 1/3 的位置是超大的面部特写，右边 2/3 放正视图、侧视图、后视图)，纯白背景，21:9 横向宽幅构图。二次元国漫风，18 岁少年剑修，高个偏瘦，身高约 178cm，肩膀不宽但站姿挺直。脸型略瘦，五官立体，眉形锋利，黑色短发略微凌乱，刘海微微遮住额头，瞳色金黄，眼神冷静中带一点若有所思。皮肤偏白但有轻微小麦色。上身黑金配色修身战斗长袍，肩部和胸口有简单金色线条装饰，下身深灰色长裤，脚穿黑色长靴。左耳戴一枚小巧银色耳钉，腰间挂着一柄造型简洁的长剑。表情平静，嘴角略微收紧，给人距离感但不阴郁。全身站立正视图，单人，白色纯色背景，21:9 横向宽幅构图，高精细线稿与上色，无道具，无其他人物，无文字。
```

## 二、场景提示词模板

### 框架
```
【空间类型】+【时代与风格】+【主要结构与物件】+【光线与时间】+【氛围与情绪】+【是否有角色】+【画面规格】
```

### 模板A：室内场景
```
空镜头，无人场景，环境概念设计图。
[风格关键词]，[空间类型]，[时间]。
[主要结构描述]，[家具与物件]，[装饰细节]。
[光线描述]，[色调]。
氛围：[情绪关键词]。
画面规格：21:9宽幅构图，高精细度，8K分辨率。
```

### 模板B：室外场景
```
空镜头，无人场景，环境概念设计图。
[风格关键词]，[场景类型]，[时间与天气]。
[地形与建筑]，[植被与自然元素]，[远景描述]。
[光影效果]，[主色调与辅助色]。
氛围：[情绪关键词]。
画面规格：21:9宽幅构图，电影级构图，8K分辨率。
```

### 示例1：古风修真场景
```
古风修真学院的练武场，外景，宽阔的石质广场，地面刻有巨大的阵法纹路，四周是高耸的石柱与旗帜，背景可以隐约看到远处的山脉和主楼。下午偏晚的阳光从左上方斜照进来，形成长长的影子，整体色调偏冷，略带一点金色余晖，营造庄严又略微压迫的氛围。画面中暂时不要出现任何人物，适合作为动画分镜的背景，21:9宽幅构图，高精细度。
```

### 示例2：自然环境
```
空镜头，无人场景，环境概念设计图。
环境：幽深茂密的竹林，清晨，雾气缭绕。地面铺满枯黄的竹叶，有一条蜿蜒的石板路延伸至远方。
氛围：静谧，神秘，略带压抑感，冷色调，青绿色系。
光影：丁达尔效应，微弱的阳光穿透竹叶投下斑驳光影。
风格：写实厚涂风格，电影级构图，8K分辨率，21:9宽幅比例
```

## 三、首帧/尾帧图片提示词模板

### 框架
```
(主体描述) + (环境场景) + (艺术风格/媒介) + (光影与色彩) + (构图与视角) + (质量修饰词)
```

### 强调语法
- 用 `()` 包裹需要强调的内容
- 多层括号表示更强的强调：`(())` > `()`

### 完整示例
```
中景镜头。图1中的主角正站在图3狭窄阴暗的雨夜小巷中，身体微侧，准备迎战。
她的右手正拔出图2那把红色等离子武士刀，刀身出鞘一半，红光照亮了她的侧脸。她冷酷严肃的表情，眼神死死盯着前方。雨水打在她的脸上和刀身上。
赛博朋克街道风格。潮湿的墙壁，乱接的电线，远处有模糊的粉色霓虹灯光。地面要有积水倒影。高对比度，电影感，赛博朋克色调
侧逆光勾勒人物轮廓。红色的刀光作为主要补光照亮面部。背景光为冷色调。体积光穿过雨丝。尺寸比例为9:16
```
