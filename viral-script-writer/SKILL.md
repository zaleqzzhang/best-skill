# 爆款口播视频脚本创作专家

## 核心原则：黄金3秒定生死

| 指标 | 数据 |
|-----|------|
| 用户判断内容时间 | < 2秒 |
| TikTok/抖音算法检查点 | 3秒 |
| 3秒留存率目标 | > 65% |
| 高留存 vs 低留存曝光差异 | 4-7倍 |

**算法逻辑**：
- 3秒内划走 → 内容不优质 → 降权
- 3秒内停留 → 内容优质 → 推荐更大流量池

## 十大钩子公式

### 1. 引发好奇
**公式**：`为什么/怎么办/如何做……？`
**示例**："为什么你的视频总是40%完播率就掉了？"

### 2. 直击痛点
**公式**：`___不敢___怎么办？ / ___经常___怎么办？`
**示例**："论文数据不敢上传ChatGPT怎么办？"

### 3. 放大焦虑
**公式**：`___都___了，你还没___`
**示例**："2026年了，你还不会用AI工具？"

### 4. 价值展示
**公式**：`价值___的___，赶紧码住！ / 这是全网最完整的___`
**示例**："价值十万的OpenClaw配置经验，赶紧码住"

### 5. 抛出利益
**公式**：`有了___，再也不用___`
**示例**："有了这个工具，再也不用手动整理文献了"

### 6. 巧用数字
**公式**：`99%的人都不知道，___ / 3个方法帮你___`
**示例**："99%的高校老师都不知道这个工具"

### 7. 警告避坑
**公式**：`千万别___，否则___ / ___一定会踩的坑`
**示例**："千万别把论文上传ChatGPT，否则后悔都来不及"

### 8. 借势权威
**公式**：`___都在用的___`
**示例**："清华教授都在用的AI工具"

### 9. 内幕揭秘
**公式**：`揭秘一个___不为人知的内幕 / ___不会告诉你的真相`
**示例**："揭秘高校AI部署不为人知的内幕"

### 10. 反向输出
**公式**：`我们都被___骗了，其实___ / ___根本就不是___`
**示例**："我们都被大厂骗了，其实开源更好用"

## VizzyLabs 三步公式

### Step 1: Context Lean（上下文铺垫）
> 前1-2行，清晰说明主题 + 建立观众关联
> **示例**："这个技术太强了..."

### Step 2: Scroll-Stopping Interjection（转折打断）
> 用"但是""然而"制造张力
> **示例**："...但最厉害的部分不是你想的那样"

### Step 3: Contrarian Snapback（反转揭示）
> 意外揭示，创造信息闭环
> **示例**："大家都盯着那块大屏幕——但真正的魔法在音频系统"

## 脚本结构模板

### 通用模板
```
【0-3秒】钩子（十大公式任选）
   ↓
【3-15秒】价值核心
   ├─ 放大痛点（让观众有共鸣）
   ├─ 解决方案（给出具体方法）
   └─ 案例佐证（真实故事/数据）
   ↓
【15秒-结束】行动召唤
   ├─ 互动类："评论区告诉我你的答案"
   ├─ 留资类："评论XX领取XXX"
   └─ 关注类："关注我，明天继续"
```

### 教育类模板
```
【钩子】直击痛点/问题钩子
【痛点】1-2句话放大痛点
【方案】给出具体解决方案（分步骤）
【证明】案例/数据支撑
【召唤】评论区留言领取资源
```

### 故事类模板
```
【钩子】爆点前置（最精彩的部分）
【背景】简短交代背景
【过程】讲述故事发展
【反转/升华】意外结局或情感升华
【召唤】关注看更多
```

## 平台差异化策略

| 平台 | 时长 | 语速 | 钩子偏好 | 关键指标 | 完播率目标 |
|-----|------|-----|---------|---------|-----------|
| **抖音** | 30-60秒 | 快（200-250字/分钟） | 情绪共鸣+金句 | 完播率、评论 | > 50% |
| **TikTok** | 15-30秒 | 中快 | 视觉冲击+音乐卡点 | 3秒留存、分享 | > 65% |
| **YouTube** | 1-3分钟 | 中等 | 深度价值+专业感 | 观看时长、订阅 | > 40% |
| **X.com** | < 30秒 | 快 | 观点犀利+时效 | 转发、引用 | > 60% |
| **B站** | 1-5分钟 | 中等 | 干货+梗文化 | 三连、弹幕 | > 35% |

### 抖音特化
- 语速：200-250字/分钟
- 结构：总-分-总 或 钩子-干货-召唤
- 音乐：配合热门BGM

### TikTok 特化
- 3秒留存必须 > 65%
- 无声友好：60%用户静音观看，确保文字可读
- 每3秒一个信息点

## 创作流程

### Step 1: 明确核心价值
> 用一句话概括：观众看完能得到什么？

### Step 2: 选择钩子类型
根据内容类型选择：
- 教育类 → 直击痛点 / 问题钩子 / 数字钩子
- 故事类 → 爆点前置 / 反转钩子
- 娱乐类 → 模式中断 / 好奇钩子
- 观点类 → 反向输出 / 借势权威

### Step 3: 写3个钩子变体
> 不要用第一个想法，写3个版本，选最好的

### Step 4: 填充主体内容
> 痛点放大 → 解决方案 → 案例证明

### Step 5: 设计行动召唤
> 明确告诉用户下一步做什么

## 避坑指南

| 错误 | 表现 | 正确做法 |
|-----|------|---------|
| ❌ 慢热开场 | "大家好，今天我要讲的是..." | 直接上钩子 |
| ❌ 通用问候 | "Hey guys, welcome to my channel" | 删掉问候，直接开始 |
| ❌ 埋没亮点 | 最精彩的内容在第15秒 | 爆点前置 |
| ❌ 钩子不匹配 | 承诺的内容与实际不符 | 确保承诺可兑现 |
| ❌ 过度标题党 | 钩子夸张，内容平庸 | 诚实但有吸引力 |
| ❌ 无行动召唤 | 结尾没有引导 | 明确告诉用户下一步 |
