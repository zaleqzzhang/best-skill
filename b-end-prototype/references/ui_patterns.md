# UI 设计模式库

> 本文件提供与设计系统无关的通用 UI 设计模式知识，帮助识别参考图中的页面类型和交互模式。

## 页面类型识别

| 页面类型 | 典型特征 | 核心组件 |
|---------|---------|---------|
| 列表页 | 数据表格/卡片列表 + 操作栏 + 分页 | 筛选区、数据区、批量操作、分页器 |
| 表单页 | 标签 + 输入控件 + 提交按钮 | 表单分组、验证反馈、操作栏 |
| 详情页 | 基本信息 + 内容分区 | 信息卡片、Tab 页签、关联数据 |
| 仪表盘 | 数据卡片 + 图表 | 统计卡片、图表组件、时间筛选 |
| 配置页 | 配置项分组 + 开关/选择器 | 配置分组、状态切换、保存操作 |

## 导航模式

- **侧边导航**：中后台系统最常用，左侧固定菜单，支持多级展开
- **顶部导航**：适用于功能模块较少的场景，水平排列
- **Tab 导航**：同一页面内切换不同内容板块
- **面包屑**：展示页面层级路径，支持逐级返回
- **分页器**：大量数据的分页浏览

## 内容展示模式

- **表格视图**：结构化数据展示，支持排序/筛选/多选
- **卡片视图**：信息聚合展示，适合内容丰富的条目
- **列表视图**：简洁的条目展示，适合信息密度较低的场景
- **网格视图**：等宽等高的网格排列，适合图片/资源浏览
- **时间轴**：时序信息展示，适合日志/历史记录

## 表单设计模式

- **水平布局**：标签在输入框左侧，节省垂直空间（B 端常用）
- **垂直布局**：标签在输入框上方，扫描效率高
- **卡片式分组**：相关字段用卡片容器分组
- **分步表单**：复杂表单拆分为多个步骤

## 操作模式

- **单条操作**：行内操作按钮（编辑/删除/更多）
- **批量操作**：复选框 + 批量操作条
- **确认弹窗**：危险操作前的二次确认
- **表单弹窗**：轻量级新建/编辑，不跳转页面
- **抽屉面板**：右侧滑出，适合复杂表单和详情

## 搜索与筛选模式

- **搜索框**：关键词搜索，可带搜索建议
- **筛选标签**：已选筛选条件以标签形式展示，可逐个移除
- **下拉筛选**：单个维度的筛选选择
- **高级筛选**：多维度组合筛选，可折叠展开

## 反馈与状态模式

- **加载态**：骨架屏 / Loading 动画 / 进度条
- **空态**：插图 + 提示文字 + 引导操作按钮
- **错误态**：错误说明 + 重试按钮
- **成功反馈**：Toast 消息提示 / 状态更新
- **状态标签**：用颜色区分不同状态（成功/警告/错误/进行中等）
