## Centered Symmetrical

### Description
Subject placed dead-center with balanced left/right mirroring. Creates iconic, memorable, bold impact.

### When to Use
- Heroic characters or objects
- Frontal confrontation moments
- Meditative or spiritual themes
- Maximum brand recognition

### Prompt Keywords
```
centered symmetrical composition, balanced mirrored layout, frontal iconic
placement, dead-center focus, perfect vertical axis symmetry
```

### Examples

**Hero Portrait:**
```
Masked vigilante face centered perfectly in Mondo poster style, symmetrical
composition, balanced left-right elements, 3-color print: black, red, cream,
frontal iconic hero placement, vertical axis mirror symmetry, vintage superhero
poster aesthetic
```

**Symbolic Object:**
```
Glowing sword pointing upward centered in Mondo poster style, perfect vertical
symmetry, geometric rays emanating balanced left-right, 2-color print: gold and
deep blue, dead-center focal power, minimalist legendary weapon iconography
```

## Negative Space Mastery

### Description
Strategic use of empty space to create hidden imagery, imply scale, or craft visual double-meanings. Empty areas are as important as filled ones.

### When to Use
- Minimalist designs
- Clever conceptual posters
- Films with mystery or hidden themes
- Limited color palettes (1-3 colors)

### Prompt Keywords
```
negative space storytelling, empty space creates hidden image, clever use of
blank areas, visual double meaning, what's not shown tells the story, silhouette
reveals secondary image
```

### Techniques

**Hidden Image:**
Subject silhouette contains a completely different image within the negative space.

```
Detective silhouette in fedora, negative space within hat and shoulders forms
city skyline in Mondo poster style, clever dual imagery, 2-color print: black
and cream, Olly Moss negative space technique, what's missing creates secondary
story
```

**Implication Through Absence:**
Show only part of something, let negative space imply the rest.

```
Single shark fin breaking water surface, vast negative space ocean in Mondo
poster style, minimalist threat implication, 2-color print: ocean blue and white,
what's beneath unseen creates tension, empty space suggests lurking danger
```

## Geometric Framing

### Description
Using circles, triangles, arches, or other geometric shapes to frame or contain the main imagery.

### When to Use
- Organized composition needs
- Art Deco or period styling
- Portal/window metaphors
- Visual hierarchy control

### Prompt Keywords
```
geometric [circle/triangle/arch/hexagon] framing, contained within [shape],
border created by [geometric form], Art Deco decorative geometry, structured
compositional boundaries
```

### Shape Meanings

**Circle:** Unity, wholeness, cycles, completeness
**Triangle:** Conflict, hierarchy, ascension, instability
**Arch:** Gateway, passage, Gothic grandeur, threshold
**Rectangle/Border:** Traditional, formal, gallery-style
**Hexagon:** Technical, modernity, efficiency

### Examples

**Circular Portal:**
```
Spaceship emerging from circular tunnel in Mondo poster style, perfect circle
geometric framing, concentric rings creating depth, 3-color print: navy, silver,
orange accent, centered radial composition, contained within spherical boundary,
portal metaphor
```

**Gothic Arch:**
```
Vampire castle framed within Gothic pointed arch in Mondo poster style, ornate
Art Deco arch border, vertical symmetry within arch shape, 4-color print: black,
burgundy, gold, cream, architectural decorative geometry, threshold to horror realm
```

## Layered Depth

### Description
Distinct foreground, midground, and background layers creating dimensional depth despite flat screen print aesthetic.

### When to Use
- Landscape-heavy designs
- Epic scale emphasis
- Environmental storytelling
- Western, sci-fi, fantasy genres

### Prompt Keywords
```
layered depth with distinct foreground/midground/background, three-plane
composition, foreground silhouette against mid-distance [element] and far
[element], dimensional layers in flat screen print, overlapping planes create
depth
```

### Layer Strategies

**Dark → Mid → Light:**
Black foreground silhouette, medium-tone midground, pale background

```
Black twisted tree in foreground, fog-shrouded mansion in midground, pale moon
background in Mondo poster style, three distinct depth layers, 3-color print:
black, gray, cream, overlapping planes, dimensional despite flat colors
```

**Scale Through Layers:**
Small foreground, larger midground, massive background

```
Tiny astronaut in foreground, medium rover in midground, massive planet looming
in background in Mondo poster style, layered scale progression, 3-color print:
rust orange, teal, black, depth through size relationships
```

## Silhouette Dominance

### Description
Subjects reduced to pure silhouettes without internal detail - shape, posture, and outline tell the story.

### When to Use
- Iconic character recognition
- Emphasizing mood over details
- Limited color palettes
- Mystery and atmosphere
- Classic noir, westerns, minimalism

### Prompt Keywords
```
pure silhouette without internal detail, filled black shape, recognizable by
outline only, no facial features shown, posture and shape tell story, iconic
profile silhouette
```

### Silhouette Types

**Character Recognition:**
Shape alone identifies the character (cape, hat, weapon stance)

```
Batman cape and cowl silhouette, no face details, pure black shape against
cream background in Mondo poster style, recognizable by iconic outline only,
2-color print, posture conveys vigilante stance
```

**Environmental Mood:**
Landscape or architecture as silhouette

```
Victorian mansion pure black silhouette against blood red sunset in Mondo poster
style, no architectural details just gothic outline, 2-color print, ominous
shape creates foreboding mood
```

**Layered Silhouettes:**
Multiple overlapping silhouetted forms

```
Cowboy on horseback layered over desert mesa layers, all in silhouette in Mondo
poster style, overlapping black shapes, 3-color print with sunset gradient behind,
depth through silhouette stacking, no internal details needed
```

## Visual Puns & Hidden Imagery

### Description
Clever compositions where one object doubles as another, or hidden imagery rewards close inspection. Laurent Durieux specialty.

### When to Use
- Cerebral or twist-ending films
- Showing cleverness and craft
- Films with double meanings
- Creating collector appeal (discovery factor)

### Prompt Keywords
```
visual pun where [A] doubles as [B], hidden [element] within [main subject],
clever dual-purpose composition, secondary imagery rewards inspection, Laurent
Durieux style visual trick, object serves double meaning
```

### Techniques

**Object Transform:**
One object cleverly transitions into or contains another

```
Opened briefcase transforms into city skyline in Mondo poster style, visual pun
composition, business case lid becomes building skyline, clever dual-purpose
imagery, 3-color print: navy, bronze, cream, corporate world doubles as urban
landscape
```

**Hidden Figure:**
Face or form concealed within larger composition

```
Forest landscape with Bigfoot face hidden in tree line in Mondo poster style,
secondary creature imagery rewards inspection, 4-color print: forest greens and
browns, Laurent Durieux subtle hidden figure, mystery revealed on close viewing
```

**Metaphorical Fusion:**
Two concepts merged into single image

```
Brain cross-section that's also a maze in Mondo poster style, psychological
metaphor visual, mind-as-puzzle imagery, 2-color print: pink and cream, clever
conceptual fusion, represents psychological thriller themes
```

## Typography Integration

### Description
Hand-drawn retro lettering as compositional element, not just text overlay. Typography participates in the design.

### When to Use
- Title is short and iconic
- Retro period accuracy needed
- Text is part of the visual story
- Emphasizing graphic design heritage

### Prompt Keywords
```
bold retro typography integrated into composition, hand-drawn lettering as
design element, [Art Deco/condensed sans-serif/hand-painted] title treatment,
vintage [decade] lettering style, typography participates in visual design
```

### Typography Styles by Era

**1940s-50s:** Bold condensed sans-serifs, hand-painted rounded letters
**1960s:** Psychedelic flowing curves, Op-Art geometric fonts
**1970s:** Thick geometric block letters, disco-influenced
**1980s:** Chrome effects, italic speed lines, neon outlines

### Integration Techniques

**Text as Shape:**
Letters form silhouettes or frame imagery

```
Film title letterforms create city skyline shape in Mondo poster style, typography
as architectural composition, bold condensed Art Deco letters, 3-color print,
text participates as visual element not overlay
```

**Through-Line Typography:**
Text follows compositional flow or object contours

```
Vintage 1950s hand-painted script title curves along winding road in Mondo poster
style, typography follows landscape flow, integrated lettering, warm earth tones,
text is part of journey composition
```

**Geometric Letter Blocks:**
Thick block letters as design foundation

```
Bold 1970s block letters "ESCAPE" with prison bars as letter strokes in Mondo
poster style, typography conveys theme structurally, 3-color print: orange, brown,
cream, letters are compositional architecture
```

## Border & Edge Treatments

### Description
Intentional frame designs, decorative borders, or unconventional edge treatments that enhance the poster's collectibility and print aesthetic.

### When to Use
- Period-accurate styling (Art Deco, Victorian)
- Gallery/art print emphasis
- Adding ornamental detail
- Framing chaotic center composition

### Prompt Keywords
```
[Art Deco/ornate/minimalist] decorative border, edge treatment with [pattern],
ornamental frame design, vintage poster border styling, structured edge containing
composition, gallery print frame aesthetic
```

### Border Styles

**Art Deco:** Geometric step patterns, radiating lines, symmetrical ornaments
**Victorian/Gothic:** Ornate flourishes, filigree, archways
**Minimalist:** Simple ruled lines, corner marks
**Grunge:** Torn edges, distressed borders, print bleed
**None/Full Bleed:** Image extends to edges, modern gallery approach

### Examples

**Ornate Frame:**
```
Horror scene within ornate Art Deco border in Mondo poster style, geometric
decorative frame with corner flourishes, 4-color print: black, gold, burgundy,
cream, edge treatment enhances 1920s period accuracy, contained composition
```

**Distressed Edges:**
```
Action scene with intentionally rough torn edges in Mondo poster style, distressed
border simulating aged print damage, 3-color screen print with edge texture,
vintage weathered aesthetic, grunge border treatment
```

**Gallery Border:**
```
Centered image with wide minimalist border in Mondo poster style, clean ruled
line frame, ample negative space margin, 2-color print: image content on cream
background, gallery exhibition aesthetic, museum print presentation
```

## Combining Composition Patterns

Most effective Mondo posters layer multiple compositional techniques:

**Example - Centered + Negative Space + Geometric:**
```
Astronaut helmet centered in circular frame, negative space within helmet visor
shows alien landscape reflection in Mondo poster style, combined techniques:
centered symmetry plus geometric circular framing plus negative space storytelling,
3-color print, multiple compositional layers working together
```

**Example - Layered Depth + Silhouette + Border:**
```
Three-layer silhouette composition with ornate Art Deco border in Mondo poster
style, foreground gunslinger silhouette, midground desert, background mountains,
all framed in geometric decorative border, 4-color print, combining depth
layering with silhouette approach and period border treatment
```

## Composition Checklist

When crafting Mondo prompts, consider:

- [ ] Is there a clear focal point?
- [ ] Does negative space serve a purpose?
- [ ] Would geometric framing strengthen it?
- [ ] Are depth layers distinct?
- [ ] Does silhouette approach fit the subject?
- [ ] Is there opportunity for visual puns?
- [ ] Should typography integrate structurally?
- [ ] Would a border enhance the period accuracy?

## Anti-Patterns to Avoid

**Don't:**
- Center everything without purpose (boring symmetry)
- Fill every inch unless doing Tyler Stout maximalism
- Use photorealistic details in silhouettes (breaks the aesthetic)
- Add borders just because (they should serve the design)
- Hide imagery so cleverly it's invisible (visual puns should reward, not frustrate)
- Ignore composition in favor of just "Mondo style" (composition IS the style)

**Do:**
- Choose composition that serves the film's theme
- Let empty space breathe
- Make deliberate framing choices
- Layer techniques for sophisticated results
- Study real Mondo examples for compositional inspiration
