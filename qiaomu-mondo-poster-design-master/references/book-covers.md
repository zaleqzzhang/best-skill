## Format Considerations for Books

### Key Differences from Movie Posters

**Aspect Ratio:**
- Standard book: 6x9" portrait (2:3 ratio)
- Trade paperback: Taller, narrower than posters
- Hardcover dust jacket: Includes spine and back

**Typography:**
- Title must be highly legible (smaller format)
- Author name is critical (not optional like movie posters)
- Genre signals through typography choices

**Shelf Presence:**
- Must work at thumbnail size (online stores)
- Spine design crucial for bookstore display
- Competition with thousands of other covers

### Mondo Adaptations for Books

**Simplify further:**
- Even more minimalist than movie posters
- Single strong focal element
- High contrast for visibility

**Legible typography:**
- Larger, bolder text treatment
- Author name as prominent as title
- Clear hierarchy

**Vertical composition:**
- Embrace portrait orientation
- Use vertical lines and elements
- Top-to-bottom visual flow

## Book Cover Checklist

When designing Mondo book covers, ensure:

- [ ] Title is legible at thumbnail size
- [ ] Author name is prominent and clear
- [ ] Portrait orientation optimized
- [ ] Single strong focal element
- [ ] Limited color palette (2-4 colors)
- [ ] Genre appropriately signaled
- [ ] Symbolic rather than literal imagery
- [ ] Screen print aesthetic maintained
- [ ] Typography hierarchy clear
- [ ] Works in both color and grayscale
- [ ] Spine design considered (if applicable)
- [ ] Competitive analysis vs. similar titles

## Typography Best Practices

### Title Treatment
- **Literary Fiction**: Elegant serifs, restrained
- **Sci-Fi/Fantasy**: Bold sans-serifs or custom
- **Mystery/Thriller**: Sharp, high-contrast fonts
- **Horror**: Distressed or Gothic fonts
- **Non-Fiction**: Professional, credible fonts

### Author Name
- Always prominent, never an afterthought
- Size relationship to title: 30-50% of title size
- Position: Usually bottom, sometimes top
- Credential inclusion for non-fiction

### Hierarchy
1. Title (largest)
2. Author name (prominent)
3. Subtitle/series info (smaller)
4. Endorsements/quotes (smallest, back cover)

## Print vs. Digital Considerations

**For Print:**
- Higher detail possible
- Spine design crucial
- Paper texture simulation

**For Digital:**
- Must work at 200x300px thumbnail
- Higher contrast needed
- Simpler compositions preferred
- Bold typography essential

## Series Design

When designing multiple books in a series:
- Consistent color palette across books
- Repeated compositional structure
- Varied symbolic elements per book
- Clear series branding
- Collector appeal through consistency
