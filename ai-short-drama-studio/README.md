# AI Short Drama Studio

## 简介

`ai-short-drama-studio` 是一个面向 **短剧创作全流程** 的通用型 Skill。它能根据用户当前所处阶段，自动在 **创意提案、五维评分、剧本创作、分镜设计、全链路交付** 五种模式间切换，把一个模糊想法推进成可直接进入制作阶段的交付包。

## 参赛定位

- **技能名称**：`ai-short-drama-studio`
- **推荐分类**：**灵感创意**
- **版本**：`1.0.0`
- **技能形态**：**单技能全能版**
- **目标问题**：解决短剧从“有想法”到“能开做”之间流程断裂、判断失序、输出不可执行的问题

## 核心价值

- **一个入口，自动分流**：用户不需要记住多个命令或多个技能，直接描述需求即可
- **先判断，再细化**：先做创意筛选与优先级判断，避免在低价值题材上浪费制作成本
- **输出可执行，不止灵感**：最终交付的是提案、评分、剧本方案包、镜头参考结构与发布建议，而不是空泛点子
- **兼顾创作与制作**：既看情绪和故事，也看可拍性、成本、时长和传播潜力

## 五种模式

| 模式 | 场景 | 主要输出 |
|------|------|----------|
| **Mode A 创意提案** | 用户只有主题、情绪、关键词 | 10-15 个创意方案 + 初筛结论 |
| **Mode B 五维评分** | 用户要判断先做哪个 | 评分表 + 排序 + 投入建议 |
| **Mode C 剧本创作** | 用户已选中一个方向 | 完整剧本方案包 |
| **Mode D 分镜设计** | 用户已有剧本或分段大纲 | 镜头单元卡 + 九格视觉参考板 + 动态执行建议 |
| **Mode E 全链路模式** | 用户要求一条龙完成 | 提案 → 评分 → 剧本 → 分镜 |

## 为什么它适合比赛评审

### Value

- 解决的不是单点功能，而是短剧工作流中的 **方向判断 + 生产组织 + 输出落地** 三个关键问题
- 对内容创作者、短视频团队、AI 视频尝试者都有真实提效价值

### Usability

- 只有一个技能入口，理解成本低
- 内部按五种模式拆开，结果结构清晰，不会变成“大 prompt 乱炖”
- 输出格式可直接复制继续使用

### AI-Nativeness

- 技能不是机械填模板，而是根据输入内容自动判断用户所处阶段
- 同时把生成、筛选、诊断、结构化交付融在一个 AI 原生工作流里

### Impact

- 适用于短剧、短片、微剧情、内容实验片等多个创作场景
- 既可以服务个人创作者，也可以作为团队前期开发的标准流程

## 使用方式

1. 将 `ai-short-drama-studio` 文件夹放入 `~/.codebuddy/skills/` 或项目内 `.codebuddy/skills/`
2. 在对话中直接描述需求，例如：
   - “想几个关于代际关系的短剧方向”
   - “这 5 个创意帮我排序”
   - “把这个一句话梗概写成 5 分钟短剧”
   - “把这个剧本转成分镜手册”
   - “从选题到分镜一条龙给我做完”
3. Skill 会自动选择最适合的模式执行
