## 详细说明（可直接粘贴到“使用说明”）

`AI短剧全流程工作室` 是一个面向短剧创作者、短视频团队和 AI 视频爱好者的一站式创作 Skill，用来解决短剧开发过程中最常见的几个问题：不知道拍什么、不会判断哪个创意更值得做、很难把一句话想法扩展成完整剧本、以及缺少可直接执行的分镜方案。

这个 Skill 的核心特点不是“只会生成文本”，而是会根据用户当前所处的创作阶段，自动切换到最合适的工作模式，帮助用户把一个模糊想法逐步推进成可进入制作阶段的完整交付包。它支持五种核心能力：

1. **创意提案**：当用户只有一个主题、情绪或关键词时，自动生成多组短剧创意方案，并给出初步筛选结论。
2. **五维评分**：当用户手里有多个创意或已有剧本时，从情感张力、故事设计、主题深度、制作可行性、传播潜力五个维度进行量化评分，帮助判断“哪个更值得先做”。
3. **剧本创作**：把一句话梗概或已选中的创意，扩展为结构完整、可继续制作的短剧剧本。
4. **分镜设计**：将完整剧本进一步转化为更接近制作语言的分镜手册，包含段元信息、镜头组织、运镜建议和负面排除提示词。
5. **全链路模式**：从选题、筛选、评分、剧本到分镜一条龙完成，适合希望快速验证一个短剧方向的用户。

相比传统“一个 Prompt 只做一步”的工具，这个 Skill 更强调“先判断，再细化”。它不会盲目把所有想法都机械展开，而是优先帮助用户做创意筛选和优先级判断，避免在低价值、低可拍性的方向上浪费时间和制作成本。最终输出的也不是零散灵感，而是可以直接继续推进的结构化结果，例如创意清单、评分表、完整剧本、分镜手册和发布建议。

使用方式非常简单，用户不需要记住复杂命令，只需要直接说出自己的需求，例如“帮我想几个关于亲情的短剧方向”“这几个创意帮我排序”“把这个梗概写成 5 分钟短剧”“把这个剧本转成分镜手册”，Skill 就会自动识别当前阶段并切换模式执行。

本 Skill 适合以下场景：个人短剧创作、AI 视频前期开发、短视频团队选题会、实验短片策划、微剧情内容孵化等。它特别适合那些已经有创作想法，但缺少系统化推进流程的用户。

系统要求方面，本 Skill 以 Markdown 文档形式交付，可直接放入 CodeBuddy Skills 目录中使用，不依赖私有账号、密钥或外部专有环境，便于评审和普通用户快速安装体验。
