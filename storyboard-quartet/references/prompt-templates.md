# 单图提示词模板与示例

## 推荐公式

```
[人物身份 + 年龄感 + 关键外观锚点 + 服装]，[正在进行的动作]，[表情/情绪]，[场景空间细节]，[关键道具状态]，[光线/天气/空气感]，[景别与镜头角度]，[构图重点]，[连续性锚点]，[风格基准]，[质量标签]，[负面约束]
```

## 示例 1：写实都市剧

```
三十岁左右的短发职场女性，身穿灰色西装外套与白衬衫，右手推开玻璃门，表情疲惫但带着微微的如释重负，城市街道黄昏，身后是亮着灯的写字楼大堂，左手提着一个黑色公文包，暖黄色的夕阳从右侧打来在脸上形成侧光，中景正面微偏右构图，人物位于画面左三分之一处，门框形成画中画，与上一格办公室场景的空间方向一致，写实都市风格，cinematic still, natural light, realistic texture, clear composition, no subtitles, no watermark, no distorted hands
```

## 示例 2：温情家庭片

```
六十多岁的银发老人，穿着深蓝色毛衣和围裙，双手端着一碗热汤从厨房走向餐桌，温和的笑容，乡村厨房暖色调木质内饰，桌上已摆好两副碗筷，窗外是傍晚的田园景色，灶台上还冒着蒸汽，室内暖黄灯光从头顶吊灯洒下，中景跟随构图从左向右，人物居中偏左，热汤的蒸汽作为视觉引导，与格1中进门场景方向衔接，温情现实主义风格，cinematic still, natural light, detailed scene, realistic texture, no text overlay, no watermark, no character inconsistency
```

## 示例 3：悬疑/紧张片段

```
二十五岁左右的卷发年轻人，穿着深色连帽卫衣，蹲在巷子角落背靠砖墙，紧张地侧头窥探巷口方向，昏暗的城市小巷，地面有水渍反光，右手攥着一个折叠的信封，头顶一盏路灯发出惨白的冷光在墙面投下硬阴影，近景低角度仰拍，人物占画面右侧三分之二，巷口的光源形成纵深引导，与上一格奔跑方向保持一致，都市悬疑风格，cinematic still, clear composition, realistic texture, detailed scene, no subtitles, no watermark, no duplicated props
```

## 质量标签池

按需选择 3-5 个：

| 标签 | 适用场景 |
|------|------|
| cinematic still | 通用，几乎所有场景 |
| natural light | 自然光场景 |
| realistic texture | 需要质感的场景 |
| clear composition | 构图清晰的需求 |
| detailed scene | 场景细节丰富 |
| dramatic lighting | 戏剧性光影 |
| soft focus background | 需要虚化背景时 |
| warm color palette | 暖色调场景 |
| cool color palette | 冷色调场景 |
| high contrast | 强对比场景 |

## 负面约束池

按需选择 3-5 个：

| 约束 | 何时使用 |
|------|------|
| no subtitles | 始终建议使用 |
| no watermark | 始终建议使用 |
| no text overlay | 始终建议使用 |
| no character inconsistency | 多角色或连续场景 |
| no duplicated props | 有明确道具的场景 |
| no distorted hands | 手部特写或动作场景 |
| no extra characters | 限定人数的场景 |
| no anachronistic elements | 年代戏 |
