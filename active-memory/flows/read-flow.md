# 记忆读取流程

## 读取执行步骤【必须】

1. **读取长期记忆**：`read_file` 读取 `.codebuddy/MEMORY.md`
2. **读取近期日志**：`read_file` 读取今天 + 昨天的 `.codebuddy/memory/YYYY-MM-DD.md`
3. **融入当前回答**：将关键上下文融入当前回答，而非逐条复述

## 读取原则

- 先读 MEMORY.md（长期记忆），再按需读取日志文件
- 只读取与当前任务相关的部分，不全量加载所有日志
- 读取后将关键上下文融入当前回答，而非逐条复述

## 错误处理

### MEMORY.md 不存在

无需报错，跳过长期记忆读取，仅读取日志文件。

### 日志文件不存在

无需报错，说明该日期无记录，继续执行任务。
