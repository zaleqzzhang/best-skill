# 视频剪映手

基于 FFmpeg 的终端视频编辑助手，你的私人剪辑师——用自然语言描述需求，自动构建、解释并执行精确的 FFmpeg 命令。

你只需描述想要的效果（"去掉片头"、"加个电影感滤镜"、"把这几段视频用渐变拼起来"），视频剪映手会自动构建精确的 FFmpeg 命令，解释它的作用，经你确认后执行。

## 项目结构

```
视频剪映手/
├── SKILL.md                        # 主技能文件（工作流 + 操作索引）
├── references/
│   ├── core-editing.md             # 转码、裁剪、拼接、分割
│   ├── audio.md                    # 提取、替换、混音、音量、音效
│   ├── visual.md                   # 缩放、裁切、旋转、变速、调色、防抖、降噪
│   ├── compositing.md              # 水印、字幕、画中画、布局、绿幕、转场、特效
│   ├── export.md                   # GIF、缩略图、幻灯片、画质、帧率、元数据
│   ├── capture-streaming.md        # 录屏、摄像头、RTMP 推流、HLS/DASH、拉流
│   └── power-user.md              # 批量处理、硬件加速、倒放、章节、高级编码
├── evals/
│   └── evals.json                  # 测试用例
└── README.md
```

## 工作流程

```
你描述需求 → 剪映手理解意图 → 构建 FFmpeg 命令 → 解释命令作用 → 你确认 → 执行
```

## 功能覆盖

**30+ 类操作**，横跨 6 大领域：

- **基础剪辑**：转码、裁剪、拼接、分割
- **音频处理**：提取、替换、混音、音量/响度归一化、降噪、均衡器、静音检测
- **画面调整**：缩放、裁切、旋转、变速、调色、防抖、降噪、去隔行
- **合成与特效**：水印、字幕、画中画、网格/马赛克、绿幕抠像、40+ 转场效果、创意特效
- **导出与分发**：GIF、缩略图、幻灯片、码率控制、帧率转换、元数据
- **录制与推流**：录屏、摄像头、RTMP 推流、HLS、DASH、RTSP 拉流

## FFmpeg 环境

- 自动检测本地 `ffmpeg`
- 提供 macOS / Linux / Windows 安装命令
- 支持 `npx remotion ffmpeg` 作为快速备选方案（仅支持 H.264、H.265、VP8、VP9、ProRes）
