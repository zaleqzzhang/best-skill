# Eastmoney Core Fields (常用字段速查)

用于 `push2.eastmoney.com/api/qt/ulist.np/get`：

- `f12` 代码
- `f14` 名称
- `f2` 最新价
- `f3` 涨跌幅(%)
- `f4` 涨跌额
- `f17` 今开
- `f15` 最高
- `f16` 最低
- `f18` 昨收
- `f5` 成交量
- `f6` 成交额
- `f7` 振幅
- `f8` 换手率
- `f9` 市盈率(TTM)
- `f10` 量比

指数扩展字段：
- `f104` 上涨家数
- `f105` 下跌家数

备注：
- 该接口返回实时快照，盘中会快速变化。
- 复盘时固定时间戳，避免“同一报告内口径漂移”。
